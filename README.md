# 🚀 Web Layouts Mastery - Portfolio Collection

¡Bienvenido a mi colección de maquetación web! Este repositorio contiene una serie de desafíos de diseño UI/UX transformados en código real utilizando **HTML5** y **CSS3** moderno (Flexbox, Grid y SVG).

Este proyecto demuestra mi capacidad para convertir diseños complejos y abstractos en interfaces funcionales, responsivas y con código limpio de nivel **Junior Developer**.

---

## 📸 Galería de Desafíos (15 Capturas)

He dividido este proyecto en las secciones clave que representan la evolución del diseño:

### 1. Composiciones Abstractas y Colajes
* **El Resorte & Asteriscos:** Manejo de posicionamiento absoluto y capas (`z-index`) para elementos decorativos.
* **Organic Grid & Píldoras:** Uso de `border-radius` extremos para crear formas de arco y cápsulas.
* **Geometría Retro:** Estética minimalista con tipografías monoespaciadas.

### 2. Secciones Hero (Landing Pages)
* **Nature Voyage:** Implementación de textos curvos mediante `SVG textPath`.
* **Snack Time (Donuts):** Cuadrículas asimétricas escalonadas (Masonry-style).
* **Modern Architecture (Just that Simple):** Diseño editorial con grandes espacios en blanco y tipografía de alto impacto.

### 3. Dashboard & E-commerce
* **Dark Mode Power:** Interfaces oscuras con contrastes neón y desenfoques orgánicos (`backdrop-filter`).
* **Plant Care Zen:** Uso de degradados radiales para simular recortes curvos complejos.

---

## 🛠️ Tecnologías Utilizadas

* **HTML5:** Marcado semántico para mejor SEO y accesibilidad.
* **CSS3:** * **Flexbox & Grid:** Para layouts dinámicos y responsivos.
    * **Variables CSS (Custom Properties):** Para una gestión de colores profesional.
    * **Metodología BEM:** Nombramiento de clases claro y escalable.
* **SVG:** Manipulación de trazados para elementos decorativos y sellos circulares.
